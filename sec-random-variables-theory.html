    Section 2.1 Random variables: definition and examples

    Subsection 2.1.1 Random variables: the basics

    Informally, a random variable is a random number whose value is associated with the various outcome of an experiment.

    Example 2.1.1.

    Toss a fair coin three coins, so that the sample space \(\Omega\) includes outcomes like HHH, TTH, etc. One example of a random variable associated to this experiment is simply the number \(X\) of heads.
    Here’s a more sophisticated random variable. Suppose that we start with $1, double our money if we see heads and lose it all if we see tails. Then \(W\) could be our winnings after this experiment. (Notice that \(W = 8\) if we see HHH and \(=0\) otherwise.)

    Example 2.1.2.

    Suppose that my neighbor and I each leave home independently between 8 and 9am, each time equally likely, so \(\Omega = [8,9]\times[8,9]\text{.}\)
    One random variable is the time I leave: \((x,y) \mapsto x\text{.}\) Another is the time between our departures: \((x,y) \mapsto |x - y|\text{.}\)
    Now we are ready for the formal definition. Suppose that \((\Omega, \mathcal{F}, P)\) is a probability space.

    Definition 2.1.3.

    A random variable is a function \(X\colon \Omega \to \R\) satisfying the following technical assumption. For every \(a\in \R\)
    \begin{equation*} \{ w\in \Omega : X(w) \le a \} \in \mathcal{F}. \end{equation*}

    Remark 2.1.4.

    We’ll abbreviate e.g. \(\{w\in \Omega : X(w) \le a \}\) by \(\{X \le a \}\text{.}\)

    Proposition 2.1.5.

    Let \(X\) be a random variable. For all \(a,b\in\R\) the following are events (i.e., in \(\mathcal{F}\)).
    1. \(\displaystyle \{X \gt a\}\)
    2. \(\{X \lt a\}\) and \(\{X \ge a \}\)
    3. \(\{a \lt X \lt b \}\text{,}\) \(\{a \le X \lt b \}\text{,}\) etc.

    Proof.

    1. \(\{X \gt a\} = \{ X \le a \}^c\) and \(\mathcal{F}\) is closed under taking complements.
    2. \(\{X \lt a \} = \bigcup_{b\in\mathbb{Q}, \; b \lt a} \{X \le b\}\text{,}\) since \(\mathbb{Q}\) is order-dense in \(\R\text{.}\) Since \(\mathbb{Q}\) is countable and each \(\{X \le b\}\) belongs to \(\mathcal{F}\text{,}\) we can conclude that this countable union belongs to \(\mathcal{F}\) too.
    3. Exercise.

    Remark 2.1.6.

    The set \(\{X \in A\}\) is an event for any Borel set \(A\subseteq \R\text{.}\) The Borel sets are the smallest \(\sigma\)-field containing all open sets.

    Proposition 2.1.7.

    If \(X\) and \(Y\) are random variables on \((\Omega,\mathcal{F}, P)\) and \(b\in\R\text{,}\) then the following are all also random variables.
    1. \(\displaystyle b X\)
    2. \(\displaystyle X + Y\)
    3. \(\displaystyle X \cdot Y\)
    4. \(\displaystyle Z = \begin{cases} \frac{Y}{X} \amp \text{if } X \ne 0 \\ 0 \amp \text{if } X = 0 \end {cases}\)

    Proof.

    1. We need to show that \(\{bX\le a\}\) for every \(a\in \R\text{,}\) but \(\{X \le \tfrac{a}{b}\}\in \mathcal{F}\) since \(X\) is a random variable.
      The rest are similar.

    Definition 2.1.8.

    Associated to a random variable \(X\) is a cumulative distribution function (cdf) \(F\colon \R\to [0,1]\) defined by \(F(a) = P(X \le a)\text{.}\)

    Remark 2.1.9.

    The definition of “random variable” ensures that \(\{X\le a\}\in \mathcal{F}\text{,}\) so \(P(X\le a)\) is defined.

    Example 2.1.10.

    Let \(X\) be the number of heads in three flips of a fair coin. Describe its cdf.
    Solution.
    \begin{equation*} F(a) = \begin {cases} 0 &\text{if } a \lt 0 \\ \tfrac18 & \text{if } 0 \le a \lt 1 \\ \tfrac12 & \text{if } 1 \le a \lt 2 \\ \tfrac78 & \text{if } 2 \le a \lt 3 \\ 1 & \text{if } 3 \le a \end {cases} \end{equation*}

    Example 2.1.11.

    My neighbor and I leave our houses again. Let \(X\) be the time I leave my house; this can be thought of as the projection \((x,y) \mapsto x\text{.}\) Describe the cdf of \(X\text{.}\)
    Solution.
    \begin{equation*} F(a) = \begin {cases} 0 \amp \text{if } a \lt 8 \\ a-8 \amp \text{if } 8 \le a \lt 9 \\ 1 \amp \text{if } 9 \le a \end {cases} \end{equation*}

    Proposition 2.1.12.

    A cdf \(F\) for a rv \(X\) enjoys the following properties.
    1. \(\lim\limits_{a\to-\infty} F(a) = 0\) and \(\lim\limits_{a\to\infty} F(a) = 1\)
    2. \(F\) is increasing: if \(a \lt b\) then \(F(a) \le F(b)\text{.}\)
    3. \(F\) is right-continuous: \(\lim\limits_{x\to a^+} F(x) = F(a)\)
    4. \(\displaystyle P(X \gt a) = 1 - F(a)\)
    5. \(\displaystyle P(a \lt X \le b) = F(b) - F(a)\)
    6. \(\displaystyle P(X \lt a) = \lim\limits_{x\to a^-} F(x)\)
    7. \(P(X = a) = F(a) - \lim\limits_{x\to a^-} F(x)\) (that is, \(P(X = a)\) is the “jump” that \(F\) makes at \(a\))
    In fact, a function \(F\) satisfying the first three conditions of Proposition 2.1.12 must in fact be realizable as the cdf of some random variable \(X\text{,}\) as you might have the pleasure of verifying on the homework.

    Subsection 2.1.2 Discrete random variables

    Definition 2.1.13.

    The random variable \(X\) is discrete if it takes values only in some countable subset \(\{x_1,x_2,\dots\}\) of \(\R\text{.}\) In this case we defined its probability mass function (pmf) \(f\colon \R\to [0,1]\) by \(f(x) = P(X = x)\text{.}\)
    Note that the nonzero values of the pmf \(p_i = P(X = x_i)\) satisfy \(0 \le p_i \le 1\) and \(\sum_{1\le i} p_i = 1\text{.}\) In particular, at least one \(p_i\) must be positive! We will contrast this below with the situation for “continuous” random variables.

    Remark 2.1.14.

    Most—but not all!—discrete random variables we consider will have finite range. But many interesting examples have (countably) infinite range; in this case the range will usually be something like \(\mathbb{N}\text{.}\)

    Example 2.1.15.

    A simple but important random variable is one that can take only two possible values, \(0\) and \(1\text{.}\) Such a random variable \(X\) is called a Bernoulli random variable, and we write \(X \sim \Bernoulli(p)\) where \(p\) denotes \(P(X = 1)\text{.}\)
    Then \(X\) may represent the outcome of a biased coin or might be an indicator rv for some event \(E\text{:}\)
    \begin{equation*} X(w) = 1_E(w) = \begin {cases} 0 \amp w \in A^c \\ 1 \amp w \in A \end {cases} \end{equation*}
    The cdf \(F\) of a \(\Bernoulli(p)\) rv \(X\) is described as follows.
    \begin{equation*} F(x) = \begin {cases} 0 \amp x \lt 0 \\ 1-p \amp 0 \le x \lt 1 \\ 1 \amp 1 \le x \end {cases} \end{equation*}

    Example 2.1.17.

    Back to our usual example of tossing three coins, where \(X\) denotes the number of heads. Certainly \(X\) is discrete, since it has only four possible values: 0,1,2,3. This \(X\) has the following pmf.
    \begin{equation*} f_X(x) = \begin {cases} \tfrac18 \amp x = 0,3 \\ \tfrac38 \amp x = 1,2 \\ 0 \amp \text{otherwise} \end {cases} \end{equation*}

    Subsection 2.1.3 Continuous random variables

    Definition 2.1.18.

    The rv \(X\) is continuous if its cdf \(F\) can be expressed as the integral of an integrable function \(g\) as follows.
    \begin{equation*} F(x) = \int_{-\infty}^x g(u)\,du \quad \text{for all } x\in \R . \end{equation*}
    This (integrable) function \(g\) is called the probability density function (pdf) of \(X\text{.}\)

    Example 2.1.19.

    Leaving home, to be added

    Remark 2.1.20.

    1. \(X\) is called “continuous” since \(F_X\) is a continuous function. Notice that this implies that \(P(X = x) = 0\) for every \(x\in \R\text{.}\) See Item g.
    2. By the Fundamental Theorem of Calculus, \(\frac{d}{dx}F(x) = g(x)\) almost everywhere.
    3. \(P(X\in [a,b]) = F(b) - F(a) = \int_a^b g(u)\,du\text{.}\) In fact, \(P(X\in B) = \int_B g(u)\,du\) for any Borel set \(B\text{.}\) In particular, \(\int_{-\infty}^\infty g(u)\,du = 1\text{.}\)
    4. In particular, for small \(\epsilon\text{,}\) we have the following.
      \begin{equation*} P(X \in [x,x+\epsilon]) = \int_x^{x+\epsilon} g(u)\,du \approx \epsilon g(x) \end{equation*}
      Dividing each side of this approximate equality by \(\epsilon\text{,}\) we see that we are justified in calling \(f\) a “density” function. Even though there is no “point-mass” at \(x\text{,}\) there is density there.
    A continuous rv cannot be discrete, because a discrete rv must have point mass while a continuous rv cannot. But there are rvs that are neither continuous nor discrete.

    Example 2.1.21.

    Choose a real number from \(\Omega = [-1,1]\text{,}\) each equally likely. If it is negative, round it up to 0. Otherwise leave it alone. Let \(X\) be the result of this procedure. Show that \(X\) is neither discrete nor continuous.
    Solution.
    Then \(X\) cannot be continuous, for \(P(X = 0) = 1/2\text{.}\) But \(X\) can take any value in \([0,1]\text{,}\) of which there are uncountably many. So \(X\) is not discrete, either.
