Solution 1.2.6.1.

That \(A\) and \(B\) are independent is explained in the statement of the example. For \(A\) and \(C\text{,}\) first notice that \(P(A) = P(B) = P(C) = \tfrac16\text{.}\) Then notice that \(A \cap C = A \cap B\) and so we have
\begin{equation*} P(A \cap C) = P(A \cap B) = P(A) P(B) = P(A) P(C). \end{equation*}
A similar argument shows that \(B\) and \(C\) are independent.
But \(A \cap B \cap C = A \cap B\text{,}\) so the three events are not mutually independent:
\begin{equation*} P(A \cap B \cap C) = P(A \cap B) = \tfrac{1}{36} \ne (\tfrac16)^3. \end{equation*}
