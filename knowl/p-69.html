If \(X\) and \(Y\) are random variables on \((\Omega,\mathcal{F}, P)\) and \(b\in\R\text{,}\) then the following are all also random variables.
  1. \(\displaystyle b X\)
  2. \(\displaystyle X + Y\)
  3. \(\displaystyle X \cdot Y\)
  4. \(\displaystyle Z = \begin{cases} \frac{Y}{X} \amp \text{if } X \ne 0 \\ 0 \amp \text{if } X = 0 \end {cases}\)
