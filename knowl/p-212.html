Suppose that \(1_A\) and \(1_B\) are indicator random variables for events \(A\) and \(B\text{.}\) We have the following:
\begin{equation*} E(1_A) = P(A), \quad E(1_B) = P(B), \quad E(1_A 1_B) = P(A \cap B). \end{equation*}
So the covariance of \(1_A\) and \(1_B\) is given by
\begin{equation*} \cov(1_A,1_B) = P(A \cap B) - P(A) P(B) = P(B)( P(A \mid B) - P(A)). \end{equation*}
This means that \(1_A\) and \(1_B\) are ...
\begin{align*} \text{positively correlated} \amp \text{ iff } P(A \mid B) \gt P(A) ;\\ \text{negatively correlated} \amp \text{ iff } P(A \mid B) \lt P(A) ;\text { and} \\ \text{uncorrelated} \amp \text{ iff } P(A \mid B) = P(A). \end{align*}
