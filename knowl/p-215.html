It is easier (and in more complicated examples this will be truer) to use the formula, though:
\begin{equation*} E(XY) = 1\frac28 + 2\frac18 + 2\frac18 + 4\frac18 = \frac54, \end{equation*}
so
\begin{equation*} \cov(X,Y) = E(XY) - E(X)E(Y) = \frac54 - 1 = \frac14. \end{equation*}
