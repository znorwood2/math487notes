Definition 4.2.1.

The characteristic function of a random variable \(X\) is the function \(\phi_X \colon \R\to\C\) given by \(\phi_X(t) = E(e^{itX})\text{.}\)
When \(X\) is discrete with values in \(\N\text{,}\) this is \(\phi_X(t) = G_X(e^{it})\text{.}\) When \(X\) is continuous with density function \(f_X\text{,}\) this is \(\phi_X(t) = \int_\R e^{itx} f_X(x)\,dx\text{,}\) the Fourier Transform of \(f_X\text{.}\)
