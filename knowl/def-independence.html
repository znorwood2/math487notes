Definition 1.2.4.

Events \(A\) and \(B\) in a probability space are independent if \(P(A\cap B) = P(A)P(B)\text{.}\)
More generally, events \(A_i\text{,}\) \(i\in I\text{,}\) are mutually independent if for any finite subset \(J \subseteq I\) we have
\begin{equation*} P\left( \bigcap_{i\in J} A_i \right) = \prod_{i\in J} P(A_i). \end{equation*}
The \(A_i\) are said to be pairwise independent if for any distinct indices \(i,j\in I\) the two events \(A_i\) and \(A_j\) are independent.
