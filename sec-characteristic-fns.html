    Section 4.2 Characteristic functions

    Definition 4.2.1.

    The characteristic function of a random variable \(X\) is the function \(\phi_X \colon \R\to\C\) given by \(\phi_X(t) = E(e^{itX})\text{.}\)
    When \(X\) is discrete with values in \(\N\text{,}\) this is \(\phi_X(t) = G_X(e^{it})\text{.}\) When \(X\) is continuous with density function \(f_X\text{,}\) this is \(\phi_X(t) = \int_\R e^{itx} f_X(x)\,dx\text{,}\) the Fourier Transform of \(f_X\text{.}\)
    The characteristic function looks like just a mild modification of the mgf, but adding the \(i\) to the exponent makes a big difference: recall that for a real number \(t\text{,}\) \(\abs{e^{it}} = 1\text{.}\)

    Remark 4.2.2.

    Wait, how do we even make sense of Definition 4.2.1? The quantity \(e^{itX}\) is not (typically) a real number! We’re considering complex-valued random variables now. A function \(Y \colon \Omega \to \C\) is a complex-valued random variable if both \(\Re(Y) \colon \Omega \to \R\) and \(\Im(Y) \colon\Omega \to \R\) are random variables, so that \(Y = \Re(Y) + i \Im(y)\text{.}\)
    In the definition, we are considering \(Y = e^{itX} = \cos(tX) + i \sin(tX)\text{;}\) both \(\cos(tX) = \Re(Y)\) and \(\sin(tX) = \Im(Y)\) are random variables by standard closure properties of random variables. (This is really just jointly distributed random variables repackaged in a complex-numbers format.) Then the expectation of \(Y\) is defined to be \(E(Y) = E(\Re(Y)) + iE(\Im(Y))\text{.}\)

    Lemma 4.2.3.

    Suppose that \(X\) is a random variable with characteristic function \(\phi = \phi_X\text{.}\)
    1. \(\phi(0) = 1\text{,}\) and \(\abs{\phi(t)} \le 1\) for all \(t\in\R\text{.}\)
    2. \(\phi\) is uniformly continuous on \(\R\text{.}\)

    Proof.

    \(\phi(0) = E(e^0) = 1\text{.}\) We’ll prove the second part of (a) for continuous \(X\text{;}\) the discrete case is similar.
    \begin{align*} \abs{\phi(t)} \amp = \abs{E(e^{itx})} = \abs{\int_\R e^{itx} f_X(x)\,dx}\\ \amp \le \int_\R \abs{f_X(x)} \abs{e^{itx}}\,dx\\ \amp \le \int_\R \abs{f_X(x)}\,dx = 1 \end{align*}
    Part (b) requires some analysis. Fix \(t\) and \(h\text{.}\) We must show that \(\abs{\phi(t+h) - \phi(t)} \to 0\) as \(h \to 0\) in a way that is independent of \(t\text{.}\)
    \begin{align*} \abs{\phi(t+h) - \phi(t)} \amp = \abs{E(e^{itx}(e^{ihx} - 1))} \\ \amp = \abs{\int_\R e^{itx} (e^{ihx} - 1) f_X(x) \,dx }\\ \amp \le \int_\R \abs{e^{ihx} - 1} f_X(x)\,dx \\ \amp \underset{h\to 0}{\longrightarrow} 0 \end{align*}
    (To justify the last line, argue that the integrand is bounded by \(2f_X(x)\) and \(e^{ihx} - 1 \to 0\) and apply the Dominated Convergence Theorem.)

    Example 4.2.4.

    Find the characteristic functions of these random variables.
    1. constant r.v. \(X = a\) .
    2. \(\displaystyle X \sim \Bernoulli(p)\)
    3. \(X \in \{-1,+1\}\) with \(P(X = -1) = P(X = 1) = 1/2\text{.}\)
    4. \(\displaystyle X \sim\binomial(n,p)\)
    5. \(\displaystyle X \sim \Poisson(\lambda)\)
    6. \(\displaystyle X \sim \exponential(\lambda)\)
    Solution.
    The discrete examples (all but the last one) can be computed using the identity \(\phi_X(t) = G_X(e^{it})\) and our work on pgfs:
    1. \(\phi_X(t) = e^{iat}\text{,}\) which traverses the unit circle at speed \(\abs{a}\text{,}\) either CW or CCW depending on the sign of \(a\text{.}\)
    2. \(X \sim \Bernoulli(p)\text{:}\) \(\phi_X(t) = pe^{it} + 1-p\text{,}\) which traverses the circle of radius \(p\) centered at \(1-p\text{.}\)
    3. \(X \in \{-1,+1\}\) with \(P(X = -1) = P(X = 1) = 1/2\text{.}\) \(\phi_X(t) = \frac12 e^{it} + \frac12 e^{-it} = \cos(t)\text{.}\)
    4. \(X \sim\binomial(n,p)\text{:}\) \(\phi_X(t) = (pe^{it} + 1-p)^n\) by a fact that we’re about to prove or by our work on pgfs.
    5. \(X \sim \Poisson(\lambda)\text{:}\) \(\phi_X(t) = G_X(e^{it}) = e^{\lambda(e^{it} - 1)}\text{.}\)
    6. \(X \sim \exponential(\lambda)\text{:}\)
      \begin{align*} \phi_X(t) \amp = \int_0^\infty \lambda e^{-\lambda x} e^{itx} \,dx \\ \amp = \lambda \int_{x=0}^\infty e^{x(it - \lambda)}\,dx\\ \amp = \lambda \left[ \frac{e^{x(it - \lambda)}{it - \lambda}} \right]_{x=0}^\infty \\ \amp = \frac{\lambda}{\lambda - it} \end{align*}

    Example 4.2.5. The characteristic function of a normal rv.

    Suppose that \(X \sim \gauss(0,1)\text{,}\) so that the density of \(X\) is given by \(f_X(t) = \frac{1}{\sqrt{2\pi}} e^{-t^2/2}\text{.}\) First, we just apply the definition:
    \begin{equation*} \phi_X(t) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} e^{itx} e^{-x^2/2} \,dx = \int_{-\infty}^\infty e^{itx - x^2/2}\,dx . \end{equation*}
    Now we do something non-obvious: for a real number \(a\in \R\) we have the following.
    \begin{align*} \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty e^{ax - x^2/2}\,dx \amp = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty e^{a^2/2} e^{-(x-a)^2/2} \,dx \\ \amp = e^{a^2/2} \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty e^{-(x-a)^2/2} \,dx \\ \amp = e^{a^2/2} \frac{1}{\sqrt{2\pi}} \int_{u=-\infty}^\infty e^{-u^2/2} \,du \\ \amp = e^{a^2/2} \end{align*}
    We have shown that the functions
    \begin{equation*} \frac{1}{\sqrt{2\pi}} \int_{-\infty}^\infty e^{ax - x^2/2}\,dx \quad\text{and}\quad e^{a^2/2} \end{equation*}
    of a complex variable \(a\in\C\) agree for all \(a\in\R\text{.}\) Both of these functions are analytic, so they must agree for all \(a\in\C\text{,}\) in particular for all \(a = it\text{,}\) \(t\in\R\text{.}\) Thus,
    \begin{equation*} \phi_X(t) = e^{(it)^2/2} = e^{-t^2/2}. \end{equation*}
    The fact that the density function of a \(\gauss(0,1)\) random variable is an eigenfunction of the Fourier Transform is in some sense why the Central Limit Theorem is true.

    Definition 4.2.8.

    If the random variables \(X,X_1,X_2,\dots\) have distribution functions \(F,F_1,F_2,\dots\text{,}\) then we say that the sequence \((X_n)_n\) converges in distribution to \(X\) and write \(X_n \convdistr X\) if, for all \(x\in\R\) such that \(P(X = x) = 0\text{,}\) we have \(\lim_{n\to\infty} F_n(x) = F(x)\text{.}\)

    Remark 4.2.9.

    1. Recall that \(P(X = x) = 0\) iff \(F_X\) is continuous at \(x\text{.}\) All but countably many points are continuity points of \(F_X\text{,}\) and \(F_X(x) = F_Y(x)\) for all \(x\) in a cocountable set implies \(F_X = F_Y\text{.}\)
    2. This is the weakest of several notions of convergence of random variables. Notice that convergence in distribution refers only to the distribution functions, not on the actual values of the random variable. That is, if we replace e.g. \(X_n\) with an identically distributed \(X_n'\) (that are still independent), then that doesn’t change whether \((X_n)_n\) converges in distribution to \(X\text{.}\)

    Theorem 4.2.10. Central Limit Theorem.

    Let \(X_1,X_2,\dots\) be a sequence of i.i.d. random variables with common mean \(\mu\) and variance \(\sigma^2 \ne 0\text{.}\) Define \(S_n = X_1 + \cdots + X_n\) and notice that \(E(S_n) = n\mu\) and \(\var(S_n) = n \sigma^2\text{.}\) Let \(Y_n = \frac{S_n - n\mu}{\sigma\sqrt n}\) be the normalized sum, so that \(E(Y_n) = 0\) and \(\var(S_n) = 1\text{.}\)
    Then \(Y_n \convdistr Z\sim \gauss(0,1)\text{.}\)

    Remark 4.2.11.

    The point of all this \(Y_n\) business is just to arrange for the distributions to converge; the \(S_n\)s have means that go to \(\infty\text{.}\) In specific examples (see below) we will usually instaed approximate \(S_n\) by \(X \sim \gauss(n\mu, n\sigma^2)\text{.}\)

    Example 4.2.12.

    Roll a die 100 times. (We will assume that 100 is large enough for the CLT to provide a reasonable approximation.) Using the CLT we make the approximation
    \begin{equation*} S_{100} \approx N(100 \cdot \frac72, 100 \cdot \frac{35}{12}). \end{equation*}
    We can use this to approximate probabilities:
    \begin{align*} P( 300 \le S_{100} \le 400) \amp = P( \abs{S_{100} - 350} \le 50) \\ \amp = P(-50 \le S_{100} - 350 \le 50) \\ \amp =P\left( \frac{-50}{\sqrt{\frac{35}{12}}\sqrt{100}} \le \frac{S_{100} - 350}{\sqrt{\frac{35}{12}\sqrt{100}}} \le \frac{-50}{\sqrt{\frac{35}{12}}\sqrt{100}} \right) \\ \amp \approx P(-2.9277 \le Z \le 2.9277)\\ \amp = \Phi(2.9277) - \Phi(-2.9277) \\ \amp \approx 0.9966 \end{align*}
